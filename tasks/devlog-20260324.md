# Devlog — 2026-03-24

## ASR Bench v2 Complete

Benchmarked 4 ASR models across 3 languages (EN/ZH/JA) on Jetson AGX Orin 32GB.

### Key Findings
- **Qwen3-ASR-1.7B**: Best EN (1.87% WER) and ZH (1.57% CER), but poor JA (42% CER)
- **Whisper-turbo**: Best JA (26.92% CER), decent EN (2.36% WER), poor ZH (9.31% CER)
- **Qwen3-ASR-0.6B**: Best efficiency — near-Whisper EN, 4x better ZH, only 1.8GB VRAM
- **Whisper hallucinates** on silence/noise ("Thank you."); Qwen never hallucinates
- No single model wins all 3 languages

### Recommendation for Bosco
- EN/ZH meetings: Qwen3-ASR-0.6B
- JA meetings: Whisper-turbo
- Or: language-routed dual-model approach

### Technical Issues Encountered
1. **HuggingFace blocked** on Orin network — fixed with `HF_ENDPOINT=https://hf-mirror.com`
2. **PyTorch 2.5 lacks GQA** (`enable_gqa`) — monkey-patched `scaled_dot_product_attention` with K/V head expansion
3. **CER computation bug** — space-separating chars + `jiwer.cer` double-counted spaces. Fixed to use `cer` on raw strings.
4. **Package naming conflict** — local `datasets/` shadowed HuggingFace `datasets`. Renamed to `asrdatasets/`.
5. **torchcodec missing on aarch64** — downgraded HF datasets to 3.5.0 for soundfile audio decoding.
